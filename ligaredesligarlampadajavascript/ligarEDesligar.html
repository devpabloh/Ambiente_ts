Possibilidades de exibição de JavaScript

O JavaScript pode "exibir" dados de diferentes maneiras:

  • Gravando em um elemento HTML, usando .innerHTML: modifica o conteúdo HTML de um elemento específico, uma vantagem é que permite atualizar dinamicamente partes especificas da página sem recarregar a página inteira.
  • Gravando na saída HTML usando o .document.write(): Escreve diretamente no fluxo de documento da página web, a vantagem é ser simples para adicionar conteúdo durante o carregamento inicial da página, agora temos que tomar cuidado, porque se o mesmo for usado depois do carregamento da página, pode apagar todo o conteúdo existente na mesma. É geralmente desencorajado para manipulação de DOM depois que a página é carregada
  • Gravando em uma caixa de alerta, usando o .window.alert(): Exibe uma caixa de diálogo modal contendo uma mensagem e um botão OK. Bom para alertar aos usuários sobre informações ou erros. Uma implicação é que interrompe a interação do usuário com a página até que a caixa de alerta seja fechada. Pode ser intrusivo e irritante para os usuários se usado com muita frequência
  • Gravando no console do navegador, usando o .console.log(): Exibe informações no console do navegador. É útil para depuração e registro de informações durante o desenvolvimento. As mensagens são visíveis apenas para desenvolvedores e não afetam a experiência do usuário final. Não deve ser usado para fornecer FeedBack ao usuário.
